# 🚗 Auction Vehicle Platform

A modern, responsive auction-based web platform designed for listing and bidding on vehicles such as flood-damaged, commercial, and luxury cars. The project focuses on clean UI, structured layout, and performance optimization techniques.

---

## 📌 Features

* 🔹 Interactive homepage with hero section
* 🔹 Categorized vehicle listings (North, South, Flood, Luxury, etc.)
* 🔹 Live auction cards with countdown timer UI
* 🔹 "Why Choose Us" and workflow sections
* 🔹 FAQ section for user guidance
* 🔹 Newsletter subscription (PHP-based form handling)
* 🔹 Fully responsive design using Tailwind CSS

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, Tailwind CSS (CDN)
* **Icons:** Font Awesome
* **Backend (Basic):** PHP (Newsletter handling)
* **Styling:** Custom CSS + Tailwind Utility Classes

---

## ⚡ Performance Considerations

* Identified render-blocking resources (Tailwind CDN)
* External image usage (Unsplash API)
* Opportunities for optimization:

  * Lazy loading images
  * Converting images to WebP
  * Using Tailwind build instead of CDN
  * Implementing caching via Service Workers
